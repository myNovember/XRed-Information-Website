企業入口網站 EIP — 管理您的企業大小事

整合企業所有資訊系統，為員工打造統一的工作平台。具備高彈性的版面配置、嚴謹的權限管理，並完美支援行動化裝置。

企業內的資訊系統往往散落各地，同仁不僅難以尋找，更需反覆記憶不同的登入方式與操作介面。透過先宏資訊的 企業入口網站 EIP，我們為企業打造統一的數位入口，讓所有重要的日常資訊、表單與決策數據都在同一個平台上流暢運作。

// CORE_FEATURES

核心功能

hub整合資訊系統

整合企業內部各項應用系統，避免資訊散落各地。提供每日打卡紀錄、特休天數查詢，以及全公司的分機表等實用資訊，打造真正的企業統一入口。

dashboard數位儀錶板 Dashboard

每日即時更新企業內部的重要營運數據，讓高階主管登入後能第一時間掌握公司動態，加速關鍵決策流程。

drag_pan自由拖放 Drag-and-drop

員工可依個人工作需求與喜好，自由拖拉設定個人化版面 (Portlet)，將最常使用的系統與資訊放在最顯眼處，大幅提升日常工作效率。

verified_user個人化權限設定

面對龐雜的應用系統，資訊人員可為不同部門設定專屬選單。善用 Domino 的人員/群組/角色架構，確保法務、會計等不同角色登入後，只會看見與自身業務相關的系統項目。

campaign最新消息輪播

提供便利的後台維護介面發佈最新公告，支援設定上架區間與自動下架時間。若遇緊急事件，亦可透過推播功能即時通知全體同仁。

// DESKTOP_VIEW

EIP 桌面版實際畫面

高度整合的 EIP 桌面版，將公司所有重要資訊彙整於單一入口。

// MOBILE_X_COLLAB

行動化與協同作業

smartphone入口網站行動化

Xred EIP 全面採用 RWD 響應式網站設計架構，確保您在電腦、平板與手機上皆能享有最佳的閱讀體驗。

無論主管或同仁在外奔波，都能透過行動裝置隨時掌握公司最新訊息；若進一步整合 BPM 企業簽核，更可實現手機線上簽核，達到「不在辦公室，也能辦公事」的高效辦公模式。

mail郵件、行事曆與待辦事項整合

完美結合企業內部的個人電子郵件、行事曆與待辦事項 (To-do) 功能。

我們與 HCL Domino / Notes Mail、iNotes、Verse 的協同作業平台深度整合，能在 EIP 首頁直接帶出最新收件訊息與一週行程，讓您在單一頁面快速掌握每日工作節奏。
